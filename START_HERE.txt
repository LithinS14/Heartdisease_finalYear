⚡ QUICK START (TL;DR):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Get MongoDB Connection String
   → Go to mongodb.com/cloud/atlas
   → Create free account & cluster
   → Get connection string

2. Create server/.env File
   MONGODB_URI=your_connection_string_here
   JWT_SECRET=any_random_key_here
   PORT=5000
   NODE_ENV=development

3. Install & Run
   Terminal 1: cd server && npm install && npm run dev
   Terminal 2: cd client && npm install && npm run dev

4. Open Browser
   → http://localhost:5173
   → Sign up
   → Done! 🎉

Error? Check:

1. 💾 MongoDB Connection
   → Check MONGODB_URI in .env
   → Make sure cluster is running
   → Test connection string

2. 🔴 Backend Not Running
   → Open terminal
   → cd server
   → npm run dev
   → Should see "Server running on port 5000"

3. 🌐 Frontend Not Loading
   → Open new terminal
   → cd client
   → npm run dev
   → Should see "Local: http://localhost:5173/"

4. 📦 Missing Dependencies
   → npm install (in both server & client)
   → Clear npm cache: npm cache clean --force

5. 🔗 Can't Connect Frontend to Backend
   → Make sure backend is running
   → Check if backend port is 5000
   → Check browser console (F12) for errors

Full troubleshooting → SETUP_CHECKLIST.md
